üîß Troubleshooting Guide

Common Issues & Solutions

Issue 1: Template Literal Syntax Error

Symptom: SyntaxError: Unexpected token ')'

Cause: Using backticks with console.log without parentheses

Solution: Use proper template literal syntax: console.log(`${x} + ${y} = ${sum}`)

Issue 2: NaN (Not a Number) Result

Symptom: Output shows NaN instead of 90

Cause: Variables x or y contain non-numeric values

Solution: Verify variables are numbers: typeof x === 'number'

Issue 3: String Concatenation Instead of Addition

Symptom: Output shows "5040" instead of 90

Cause: Variables declared as strings: let x = "50"

Solution: Convert to numbers: let sum = Number(x) + Number(y)

Issue 4: Nothing Appears in Console

Symptom: No output visible

Cause: Console not opened or script not executing

Solution: Press F12 to open Developer Tools, check Console tab

Debugging Checklist

  • ‚úì Check browser console for error messages
  • ‚úì Verify all parentheses and brackets are properly closed
  • ‚úì Ensure backticks (`) are used for template literals, not quotes
  • ‚úì Confirm variables are declared with let/const/var
  • ‚úì Test variable values with console.log before calculations
  • ‚úì Check for typos in variable names
  • ‚úì Ensure script tag is properly placed in HTML
