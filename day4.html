2Ô∏è‚É£ CONDITIONAL STATEMENTS (if-else)

What are Conditions?

Conditions check if something is TRUE or FALSE and execute code accordingly.

Simple if-else

if (condition) {
    // code if TRUE
} else {
    // code if FALSE
}

Ladder if-else (Multiple conditions)

if (condition1) {
    // code if condition1 is TRUE
} else if (condition2) {
    // code if condition2 is TRUE
} else if (condition3) {
    // code if condition3 is TRUE
} else {
    // code if all are FALSE
}

Nested if (if inside if)

if (condition1) {
    if (condition2) {
        // both conditions TRUE
    } else {
        // condition1 TRUE, condition2 FALSE
    }
} else {
    // condition1 FALSE
}

let jackpot = 0;
let attempts = 0;
let gameActive = false;

function startNewGame() {
    jackpot = Math.floor(Math.random() * 100) + 1;

function makeGuess() {

    let guess = Number(document.getElementById('guessInput').value);
    let statusDiv = document.getElementById('gameStatus');

    if (!guess || guess < 1 || guess > 100) {
        statusDiv.textContent = 'Enter a valid number (1-100)';
        return;
    }

    attempts++;

    if (jackpot == guess) {
        statusDiv.innerHTML =
            `üéâ You WON!

        gameActive = false;

        statusDiv.textContent =
            `üìà Jackpot is GREATER than ${guess} (${attempts}/10)`;

            `üìâ Jackpot is SMALLER than ${guess} (${attempts}/10)`;

            `üíî Game Over!

9Ô∏è‚É£ SPECIAL CONCEPTS

Modulo Operator (%)

Used to find remainder. Used for even/odd check!

5 % 2 = 1 (5 √∑ 2 = 2 remainder 1) ‚Üí ODD
6 % 2 = 0 (6 √∑ 2 = 3 remainder 0) ‚Üí EVEN
10 % 3 = 1 (10 √∑ 3 = 3 remainder 1)
üí° Why use % for even/odd?
Even numbers divided by 2 have remainder 0
Odd numbers divided by 2 have remainder 1
