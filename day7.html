üÜï CREATE Command

üìå What is CREATE?

The CREATE command is used to create a new database or table. It defines the structure of data storage.

Creating Database

CREATE DATABASE tpuser;

Explanation:

CREATE DATABASE creates a new database named tpuser. This is the container for all tables and data.

Creating Table

CREATE TABLE stud8 ( id INT PRIMARY KEY AUTO_INCREMENT, name VARCHAR(50), age INT, city VARCHAR(50) );

Explanation:

  • id INT PRIMARY KEY AUTO_INCREMENT - Creates an auto-incrementing primary key (unique identifier)
  • name VARCHAR(50) - String field, max 50 characters
  • age INT - Integer field for age values
  • city VARCHAR(50) - String field for city names

‚úèÔ∏è ALTER Command

üìå What is ALTER?

The ALTER command modifies the structure of an existing table. You can add, modify, or delete columns.

Adding a Column

ALTER TABLE stud8 ADD email VARCHAR(100);

Explanation:

ALTER TABLE stud8 ADD email VARCHAR(100) adds a new column named email to the existing stud8 table with a maximum length of 100 characters.

üí° Other ALTER Operations:

Modify column: ALTER TABLE stud8 MODIFY COLUMN age VARCHAR(3);

Drop column: ALTER TABLE stud8 DROP COLUMN email;

Rename column: ALTER TABLE stud8 CHANGE COLUMN email student_email VARCHAR(100);

üóëÔ∏è DROP Command

üìå What is DROP?

The DROP command permanently deletes a table or database. This is irreversible - use with caution!

Dropping a Table

DROP TABLE stud8;

Explanation:

DROP TABLE stud8 permanently removes the stud8 table and all its data from the database.

‚ö†Ô∏è WARNING - Data Loss!

This command will permanently delete all data in the table. Make sure you have a backup before executing this command!

üí° Safer Alternative:

Use DROP TABLE IF EXISTS stud8; to avoid errors if table doesn't exist

Or use TRUNCATE TABLE stud8; to delete data but keep the structure (faster and allows undo in some cases)

‚ûï INSERT Command

üìå What is INSERT?

The INSERT command adds new records (rows) to a table. You provide the column names and values.

Inserting Single Record

Explanation:

üí° Inserting Multiple Records:

üîç SELECT Command

üìå What is SELECT?

The SELECT command retrieves data from a table. It's the most commonly used SQL command for viewing data.

1. Select All Records

SELECT * FROM stud8;

Explanation:

* means "all columns". This returns every column and every row from the stud8 table.

2. Select Specific Columns

SELECT name, city FROM stud8;

Explanation:

Returns only the name and city columns for all rows. Other columns are not displayed.

3. SELECT with WHERE Clause

SELECT * FROM stud8 WHERE city = 'Mumbai';

Explanation:

Returns only records where the city is 'Mumbai'. The WHERE clause filters results.

4. SELECT with Multiple Conditions (AND)

SELECT * FROM stud8 WHERE city='Mumbai' AND age > 18;

Explanation:

Returns records that meet BOTH conditions: city is 'Mumbai' AND age is greater than 18.

5. SELECT with ORDER BY

SELECT * FROM stud8 ORDER BY age DESC;

Explanation:

  • ORDER BY age - Sort by age column
  • DESC - Descending order (highest first). Use ASC for ascending

6. SELECT with LIMIT

SELECT * FROM stud8 LIMIT 3;

Explanation:

Returns only the first 3 records. Useful for pagination or limiting large result sets.

7. SELECT with LIKE (Pattern Matching)

SELECT * FROM stud8 WHERE name LIKE 'A%';

Explanation:

  • A% - Names starting with 'A' (% = any characters)
  • %a - Names ending with 'a'
  • _a% - Second letter is 'a' (_ = exactly one character)
SELECT * FROM stud8 WHERE name LIKE '%a'; SELECT * FROM stud8 WHERE name LIKE 'S__';

8. SELECT with Aggregate Functions

SELECT COUNT(*) FROM stud8; SELECT MAX(age) FROM stud8; SELECT MIN(age) FROM stud8; SELECT AVG(age) FROM stud8;

Explanation:

  • COUNT(*) - Total number of records
  • MAX(age) - Highest age value
  • MIN(age) - Lowest age value
  • AVG(age) - Average age

üîÑ UPDATE Command

üìå What is UPDATE?

The UPDATE command modifies existing records in a table. Always use WHERE clause to specify which records to update.

Updating a Record

UPDATE stud8 SET age = 21 WHERE name = 'Amit';

Explanation:

  • UPDATE stud8 - Specify the table
  • SET age = 21 - Set age column to 21
  • WHERE name = 'Amit' - Only for records where name is 'Amit'

üí° Updating Multiple Columns:

UPDATE stud8 SET age = 21, city = 'Delhi' WHERE name = 'Amit';

‚ö†Ô∏è CRITICAL - Missing WHERE Clause:

‚ùå WRONG: UPDATE stud8 SET age = 21; (This updates ALL records!)

‚úÖ CORRECT: UPDATE stud8 SET age = 21 WHERE name = 'Amit';

‚ùå DELETE Command

üìå What is DELETE?

The DELETE command removes records from a table. Always use WHERE clause to avoid deleting all records!

Deleting a Record

DELETE FROM stud8 WHERE name = 'Amit';

Explanation:

  • DELETE FROM stud8 - Specify the table
  • WHERE name = 'Amit' - Delete only records where name is 'Amit'

‚ö†Ô∏è WARNING - Data Loss!

‚ùå WRONG: DELETE FROM stud8; (This deletes ALL records!)

‚úÖ CORRECT: DELETE FROM stud8 WHERE name = 'Amit';

üí° Differences:

DELETE: Can be rolled back, slower, triggers fired

TRUNCATE: Cannot be rolled back, faster, no triggers

DROP: Removes entire table structure

üöÄ Advanced SELECT Queries

üìå Advanced Query Techniques

Combine multiple clauses for powerful data retrieval and analysis.

Complete Query Examples

Example 1: Multi-condition Query with Sorting

SELECT name, age, city FROM stud8 WHERE city = 'Mumbai' AND age > 18 ORDER BY age DESC;

What it does:

Finds students from Mumbai older than 18, shows only name/age/city, sorted by age (oldest first)

Example 2: Pattern Matching with Wildcards

-- Names starting with 'A' SELECT * FROM stud8 WHERE name LIKE 'A%'; -- Names ending with 'a' SELECT * FROM stud8 WHERE name LIKE '%a'; -- Names where 2nd character is any letter, 3rd is also any SELECT * FROM stud8 WHERE name LIKE 'S__';

Wildcard Patterns:

  • % = Zero or more characters
  • _ = Exactly one character
  • A% = Starts with 'A'
  • %a = Ends with 'a'
  • S__ = Starts with 'S' followed by exactly 2 characters

Example 3: Aggregate with LIMIT

-- Get top 3 youngest students SELECT * FROM stud8 ORDER BY age ASC LIMIT 3; -- Get total students SELECT COUNT(*) as total_students FROM stud8; -- Get statistics SELECT COUNT(*) as total, AVG(age) as average_age, MAX(age) as oldest, MIN(age) as youngest FROM stud8;
